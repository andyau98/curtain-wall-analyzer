系統可識別的圖紙類型

                

                    
位置圖
Setting-out / Layout Drawing
幕牆單元在立面上的位置、軸網、面板編號及整體尺寸
典型比例 1:100 ~ 1:200

                    
加工圖
Fabrication / Shop Drawing
個別面板的詳細加工尺寸、材料、焊接/切割/鑽孔標記及公差
典型比例 1:1 ~ 1:20

                    
大樣圖
Detail Drawing
局部節點放大細節：連接方式、密封防水、伸縮縫
典型比例 1:1 ~ 1:10

                    
組裝圖
Assembly Drawing
多構件的組裝關係與安裝順序，含零件編號
典型比例 1:5 ~ 1:50
